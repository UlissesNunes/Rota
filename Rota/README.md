## 🚀 ROTA
ERP inteligente para transportadoras, com automação por IA, segurança avançada e foco total em eficiência operacional.

📌 Sobre o Projeto
A ROTA é um produto da Leziert, desenvolvido para oferecer uma gestão completa e moderna para transportadoras.

O sistema atua como um ERP, centralizando processos, automatizando tarefas e garantindo controle total da operação.

🎯 Objetivo

Mais organizada

Mais automatizada

Mais segura

Mais escalável

🔐 Criptografia de dados
